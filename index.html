Technologies Covered in Training

Git

Git is a distributed Version Control System invented by Linus Torvalds.It is used for tracking changes in source code during software development.Its goals include speed,data integrity and support for distributed,non-linear workflows.Git is designed to coordinate work among programmers,but it can be used to track changes in any set of files.

GitHub

GitHub is a Git repository hosting service that provides access control and several collaberation features,such as wikis and basic task management tools for every project.GitHub provides a web-based graphical interface.

MySQL

MySQL is an open-source relational database management system based o SQL Language.The application is used for wide range of purposes incuding e-commerce,data warehousing and logging applications.However,the most common use is for the purpose of web database.

Django

Django is a Python-based free and open-source web application framework that is written in python and follows the model-template-view architectural pattern.It is maintained by Django Software Foundation,an American independent organisation established as a 501 non-profit.

Flask

Flask is a lightweight or micro web framework written in Python.It is classified as such because,it does not require particular tools or libraries.It has no database abstraction layer,form validation,or any other componenets where pre-existing third-party libraries provide common functions.It was designed to be easy to use and extend.

Android Studio

Android Studio is the official integrated development environment for Google's Android Operating System built on JetBrains' Intellij IDEA software and designed specifically for Android development.It accelerates your development and helps you bulid highest quality spps for every android device .It is Android's official IDE for Android application development.The official language for Android development is JAVA.Large parts of Android are written in JAVA and its APIs are designed to be called primarily from JAVA.
